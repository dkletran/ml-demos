Angular 8.1
Python 3.6
  backend/requirement.txt
# dev:
Valid settings.json (see settings.template.json) must be put in backend directory

## front end:
    npm intall 
    ng build [--watch]

    python manage.py migrate
## Launch in dev mode:
    python [backend/]manage.py runserver [0.0.0.0:port]
